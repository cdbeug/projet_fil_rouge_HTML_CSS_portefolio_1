Qui suis-je ?

Je suis actuellement inscrit à l'école Doranco, pour le Bachelor Concepteur/développeur (Bac+3/4) Le parcours donne droit à un titre RNCP (RNCP31678), niveau 6 Je suis à la recherche d'une entreprise un stage de 3 mois, commençant en juin. Le programme aborde notamment la conception (conduite de projet, UML) Ainsi que des langages tels que Java, SQL, Git (le gestionnaire de version), Ajax, MongoDB, React J'ai étudié Java pendant plusieurs années à l'Université, et faits de nombreux projets actuellement disponibles sur mon Github (lien sur le CV) (POO, interface graphique). J'ai déjà étudié Kotlin lors d'une séance d'initiation à l'université et ce langage m'a paru très accessible. J'ai aussi étudié Android lors de mon BTS SIO, en autodidacte en 2020. J'apporte plus de précisions sur mon parcours dans ma lettre de motivation. Par le passé, j'ai notamment fait du PHP5, MySQL, Javascript, NodeJS et CSS. J'ai déjà créé un site web complet (avec PHP3, et Wamp, ceci dit, il n'est pas en ligne, car c'était un projet fictif pour l'université). Le projet était le site fictif d'une association scolaire pour étudiants en informatique. Il était possible de s'inscrire, poster des messages sur le site. On pouvait aussi modifier nos informations personnelles en tant qu’utilisateur. Le gestionnaire du site (ou webmaster) avait la possibilité de voir des informations telles que le nombre de messages postés, la liste des messages et l'ancienneté de l'inscription d'un utilisateur. Il était possible pour les utilisateurs de commenter les messages d’un autre utilisateur (dans un fonctionnement de type forum). Les commentaires et les messages pouvaient être des photos. Le webmaster pouvait aussi supprimer des personnes du site. J'ai aussi eu à faire une page en Javascript, lors de mon stage à Canal+ en 2014. Le but était de construire un programme pour choisir les visuels des films proposés dans le catalogue VOD de Canal+. Je pense que la technologie Web était utilisée, plutôt qu'une application de bureau, pour des questions de facilité de maintenance (il s'agissait d'un programme Back-Office). Je suis enthousiaste à l'idée d'apprendre de nouvelles technologies web et de progresser dans ce domaine grâce à cd stage, et qui sait, peut-être pour plus longtemps ensuite.
